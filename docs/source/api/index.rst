Overview
--------

The library is organized into four main modules:

* **Extractors**: Low-level PDF text extraction using PyMuPDF and pdfplumber
* **Processors**: Text normalization, metadata parsing, and image analysis
* **Formatters**: Output formatting (Markdown, JSON, tables)
* **Utils**: Shared utilities for validation, caching, logging, and patterns
